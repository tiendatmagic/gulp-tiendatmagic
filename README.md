# Hướng dẫn cách sử dụng

Yêu cầu: cài đặt Nodejs, và cài đặt gulp sau khi cài đặt xong Nodejs

Link download Nodejs https://nodejs.org/en/

Cài đặt gulp:

### `npm install -g gulp`

Download file về và giải nén

Mở cmd và vào trong thư mục gulp

Gõ lệnh 
### `gulp`

Ấn ENTER là chạy
